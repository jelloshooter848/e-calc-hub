Bluebeam - Fundamentals

Comprehensive introduction to Bluebeam Revu navigation and interface mastery

Course Overview

Foundation-level 9-week course (24 hours) designed to introduce students to Bluebeam Revu's navigation systems and user interface. Perfect for beginners, this course requires Windows computer competency and provides essential skills for construction document management.

Course Duration

9 weeks (24 total hours)

Schedule

Tuesday & Thursday evenings

Course Level

Beginner (Foundation level)

Status

Available Now

Perfect for Beginners: This course is designed for those new to Bluebeam Revu and requires only basic Windows computer competency to get started.

What You'll Learn

Navigation & Interface

  • Bluebeam Revu interface orientation
  • Document navigation fundamentals
  • Menu systems and toolbar customization
  • Workspace organization and setup

Essential Functions

  • PDF viewing and manipulation
  • Basic annotation and markup tools
  • File management and organization
  • Print and export functionalities

9-Week Course Progression

1-3

Getting Started

  • • Software installation and setup
  • • Interface overview and orientation
  • • Basic navigation techniques
4-6

Core Functions

  • • Document viewing and manipulation
  • • Basic markup and annotation tools
  • • File organization strategies
7-9

Application

  • • Practical exercises and projects
  • • Workflow development
  • • Best practices and tips

Foundation for Advanced Learning: This course prepares students for more advanced Bluebeam courses like the Intro to Design program.
